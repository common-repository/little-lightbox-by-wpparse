Best WordPress lightbox plugin to open your images in a popup with elasticIn, fadeIn and changeIn animations.

== Description ==
Display images in a responsive popup with this light-weight and high customizable WordPress lightbox plugin. There is no coding needed on your side as the plugin has an options panel to customize everything.

Please visit [Little Lightbox Demo](http://www.wpparse.com/littlelightbox/).

### Features of WordPress Lightbox plugin (LittleLightbox)

*   **Fully Responsive**
*   **Fully customizable options pages**
*   **Works on all major browsers**
*   **Tested with all versions of WordPress**
*   **Image and gallery lightbox**
*   **Keyboard control for closing lighbox popup**
*   **Keyboard control for (Next, Previous) images**
*   **Supports fadeIn, fadeOut, elasticIn, elasticOut, changeIn, changeOut effects**
*   **Ability to enable loop for WordPress gallery lightbox images**
*   **Ability to set lightbox open and closing speed**
*   **Ability to set Next and Previous image animation**
*   **Ability to position title**

== Installation ==
Install using Plugins->Add New or manually upload using WordPress plugin uploader.
Activate the plugin, and go to the settings page for customization.
